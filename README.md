## Contesto

Le infografiche accompagnano una serie di quattro audio-analisi generate con **NotebookLM** (Google) e revisionate editorialmente, pubblicate nella sezione podcast di [Segnale e Rumore](https://salvofedele.substack.com) — newsletter di Salvo Fedele, pediatra.

La serie non insegue lo scandalo. Cerca di restituire al libro la lettura che merita: quella di un caso clinicamente eccezionale sulla genesi della vulnerabilità, utile a chiunque lavori a contatto con bambini e adolescenti.
